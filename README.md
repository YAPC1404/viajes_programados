# viajes_programados
Este repositorio contiene una app para programar tus viajes recurrentes hacia el trabajo, escuela, etc.. y en base a predicciones de trafico recibir notificaciones con anticipación de la hora en la que debes salir de tu ubicación.

## Características
- Programación de viajes fijos (ej. trabajo, escuela).
- Cálculo automático del tiempo de viaje utilizando la API de Google Maps.
- Notificaciones de recordatorio de salida (y alerta de "ya vas tarde").
- Almacenamiento local de los viajes para consultas futuras.

## Tecnologías
- Android Studio
- Kotlin
- Firebase (para notificaciones)
- Google Maps API

## Instalación
1. Clona el repositorio en tu máquina local:
    ```bash
    git clone https://github.com/YAPC1404/viajes_programados.git
2. Abre el proyecto en Android Studio.
3. Sincroniza el proyecto con Gradle.
4. Configura tu emulador o conecta un dispositivo Android.
5. Ejecuta la app.
